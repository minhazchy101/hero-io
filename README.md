# 🦸‍♂️ Hero.IO

Hero.IO is a modern frontend application that mimics a clean and interactive app marketplace experience. It focuses on building a scalable UI using React while integrating real-world features like dynamic routing, data visualization, and persistent state handling.

The goal of this project was to practice structuring a production-ready frontend application with reusable components, smooth UX, and efficient state-driven rendering.

---

## ⚡ Overview

Hero.IO allows users to:

- Browse and explore apps
- Search and filter in real-time
- View detailed app insights with charts
- Install and manage apps using local storage

The application emphasizes performance, responsiveness, and a clean developer-friendly architecture.

---

## 🛠 Tech Stack

- **React.js** — Component-based UI architecture
- **Tailwind CSS** — Utility-first styling
- **React Router** — Client-side routing
- **Recharts** — Data visualization
- **React Icons** — Icon system
- **React Select** — Advanced dropdowns
- **LocalStorage API** — Persistent client-side data

---

## ✨ Key Features

### 🔹 Clean UI Architecture

- Reusable and modular components
- Separation of concerns (UI, logic, data)
- Scalable folder structure

### 🔹 Routing System

- Multi-page feel using React Router
- Dynamic routes for app details
- Proper fallback handling for invalid routes

### 🔹 Real-time Search

- Instant filtering without reload
- Case-insensitive matching
- Smooth UX with conditional rendering

### 🔹 Data Visualization

- Interactive charts using Recharts
- Visual breakdown of user ratings
- Fully responsive graph rendering

### 🔹 Local Storage Integration

- Install apps with persistent state
- Prevent duplicate installations
- Remove/uninstall with UI sync

### 🔹 Sorting System

- Sort apps based on download count
- Clean dropdown interaction using React Select

### 🔹 Responsive Design

- Mobile-first approach
- Optimized for all screen sizes
- Consistent layout across devices
